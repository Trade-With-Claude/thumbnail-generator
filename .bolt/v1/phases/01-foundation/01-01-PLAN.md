# Phase 1: Foundation — Plan

## Acceptance Criteria

### AC1: Project structure exists
- Given the thumbnail-generator repo
- When running `python -c "from thumbgen.config import load_brand_config; print('OK')"`
- Then it imports successfully

### AC2: brand.json is valid and loadable
- Given brand.json with font, colors (focus/sleep/meditation), layout, and effects config
- When the config loader reads it
- Then all fields are accessible as a structured object

### AC3: Basic thumbnail generation works
- Given the CLI is run with `python generate.py --title "TEST"`
- When it executes
- Then a 1280x720 PNG is saved to `output/` with white text on a dark background

## Tasks

### Task 1: Project structure + brand config
- **files:** `requirements.txt`, `brand.json`, `thumbgen/__init__.py`, `thumbgen/config.py`
- **action:** Create requirements.txt (Pillow, Click). Create brand.json with full brand identity: font settings, 3 color presets (focus, sleep, meditation), layout options, effects config. Create config.py with `load_brand_config()` that reads and returns the config as a dict, with validation for required keys.
- **verify:** `pip install -r requirements.txt && python -c "from thumbgen.config import load_brand_config; c = load_brand_config(); print(c['presets']['focus']['accent'])"`
- **done:** false

### Task 2: Basic renderer + CLI
- **files:** `thumbgen/renderer.py`, `generate.py`
- **action:** Create renderer.py with `render_thumbnail(title, preset, output_path)` that creates a 1280x720 dark canvas, draws bold white text centered on it using Pillow's default font (bundled fonts come in phase 4), and saves as PNG. Create generate.py as Click CLI entry point with `--title` (required), `--preset` (default "focus"), `--output` (default "output/") options. Creates output dir if needed.
- **verify:** `python generate.py --title "ADHD FOCUS" && python -c "from PIL import Image; img = Image.open('output/thumbnail.png'); assert img.size == (1280, 720); print('OK')"`
- **done:** false

## Boundaries
- **DO NOT CHANGE:** `.bolt/` files (except STATE.md updates)
- **OUT OF SCOPE:** Text effects (glow, shadow), vignette, themes, variations, custom backgrounds, bundled fonts — all later phases
