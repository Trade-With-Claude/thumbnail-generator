# Phase 2: Text & Effects — Plan

## Acceptance Criteria

### AC1: Text has glow and shadow effects
- Given a thumbnail is generated with effects enabled in brand.json
- When viewing the output
- Then text has a visible outer glow (colored per preset) and a drop shadow

### AC2: Background has vignette and gradient overlay
- Given a thumbnail is generated
- When viewing the output
- Then edges are darkened (vignette) and a subtle gradient overlay is applied

### AC3: Color presets change the look
- Given thumbnails generated with `--preset focus` vs `--preset sleep` vs `--preset meditation`
- When comparing the three outputs
- Then each has distinct accent colors matching brand.json

## Tasks

### Task 1: Text effects + background effects modules
- **files:** `thumbgen/text.py` (new), `thumbgen/effects.py` (new)
- **action:** Create `text.py` with `draw_title(img, text, font, position, config, preset)` that draws text with drop shadow and outer glow. Create `effects.py` with `apply_vignette(img, strength)` and `apply_gradient_overlay(img, color)`.
- **verify:** `python -c "from thumbgen.text import draw_title; from thumbgen.effects import apply_vignette, apply_gradient_overlay; print('OK')"`
- **done:** false

### Task 2: Integrate into renderer + preset switching
- **files:** `thumbgen/renderer.py`
- **action:** Refactor `render_thumbnail()` to use text.py and effects.py. Apply effects in order: background tint → gradient overlay → vignette → text shadow → text glow → text. Preset switching changes glow color, background tint, and gradient.
- **verify:** `python generate.py --title "ADHD FOCUS" --preset focus && python generate.py --title "DEEP SLEEP" --preset sleep && python -c "from PIL import Image; a=Image.open('output/thumbnail.png'); print(f'Generated: {a.size}')"`
- **done:** false

## Boundaries
- **DO NOT CHANGE:** `brand.json`, `thumbgen/config.py`, `generate.py`
- **OUT OF SCOPE:** Themes, background images, variations, bundled fonts, subtitle text
